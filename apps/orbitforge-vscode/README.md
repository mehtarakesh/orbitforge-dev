# OrbitForge VS Code Extension

OrbitForge brings the same provider-agnostic coding workflow into VS Code.

## Commands

- `OrbitForge: Open Panel`
- `OrbitForge: Explain Selection`
- `OrbitForge: Plan From Workspace`

## Supported providers

- Ollama
- LM Studio
- Anthropic
- OpenAI
- OpenRouter
- Any OpenAI-compatible endpoint
